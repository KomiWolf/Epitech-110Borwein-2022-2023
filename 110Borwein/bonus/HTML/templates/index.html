110borwein :



Histoire :

110borwein est un projet de mathématique à Epitech qui consiste à calculer
la valeure de "pi/2" avec 3 méthodes différentes. la première s'intitule Midpoint
la seconde se nomme Trapezoidal
la dernière est Simpson.
Toutes ont un point commun, c'est qu'avec une formule supplémentaire,
elles sont capable de donné une valeur de "pi/2" avec beaucoup de précision.
La formule les reliants toutes se nomme "Borwien"
